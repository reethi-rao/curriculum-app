# Curriculum App — Special Needs Student Learning Portal

A calm, easy-to-navigate web app that lets special needs students access their curriculum (basic Python programming and Wix website development) and turn in their homework — designed for laptop use.

**Live demo:** add your GitHub Pages URL here once deployed, e.g. `https://<username>.github.io/<repo>/`

## What it does

- **Home** — a friendly dashboard showing what's next and quick progress on each course
- **My Courses** — switch between the two subjects (Python Programming and Website Development)
- **Lessons** — one lesson at a time, with plain-language explanations, a simple visual diagram where helpful, and step-by-step checkboxes
- **Turn in homework** — upload a file and add an optional note to the teacher
- **To-do** — everything still due, in one simple list

## Designed for accessibility

This app was built specifically for special needs students, with:

- **Reading support** — Lexend and Atkinson Hyperlegible, two typefaces designed for readability, at generous sizes throughout
- **Focus support** — one task per screen, minimal distractions, clear back buttons
- **Motor support** — large buttons and click targets (60px+ minimum)
- **Cognitive load support** — plain language, short steps, simple rounded icons, gentle progress indicators (checkmarks, not scores or streaks)
- **Calming visuals** — a soft, low-saturation teal-and-sage color palette on an off-white background

## Running it locally

Just open `index.html` in any web browser — no build step, no server, no install required.

## Deploying to GitHub Pages

1. Upload `index.html` to this repository (root folder)
2. Go to **Settings → Pages**
3. Under **Source**, choose **Deploy from a branch**, branch `main`, folder `/ (root)`, then **Save**
4. Your site will be live at `https://<username>.github.io/<repo>/` within a minute or two
